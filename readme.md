# node-ts-boilerplate

Simple express server in Typescript

Start the server based on dist folder
```sh
npm run build
```

Start the server based on dist folder
```sh
npm run start
```

Start the server for dev
```sh
npm run start:dev
```

Run unit tests
```sh
npm run test:unit
```

Run e2e tests
```sh
npm run test:e2e
```
